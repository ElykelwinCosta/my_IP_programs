Calculadora de gastos semanais, recebe uma quantidade de gastos,
valor e categoria de cada gasto, imprime quanto foi gasto com cada categoria,
imprime o percentual gasto com cada categoria em relação ao valor total.

Recebe do usuário a quantidade de gastos;
Recebe o valor gasto em cada compra;
Recebe a categoria do gasto;
Imprime o valor gasto em cada categoria;
Imprime o total gasto;
Imprime o percentual gasto em cada categoria.
